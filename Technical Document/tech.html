Javascript Loop Technical Document

FOR FOR IN FOR OF WHILE DO WHILE

FOR

A For Loop consists of three optional expressions that are enclosed in parentheses, separated by semicolons, with some statement in the loop.

Example and Uses

          
            for (let i = 0; i < 7; i++) {
              console.log(i);
            }
  • Used this loop to iterate through an array.

FOR IN

Loops through properties of objects.

Example and Uses

          
            let person = {firstname: "Dave", lastname: "Jones", age: 35};

            let text = '';
            let x;
            for (x in person) {
              text =+ person[x];
            }
  • Should not be used to iterate over an Array where the index order is important.

FOR OF

Loops over data structures that are iterable suchs as Arrays, String, Map etc.

Example and Uses

          
            let car = ['BMW', 'Honda', 'Toyota', 'Ferrari'];
            let x;

            for (x of car) {
              console.log(x + '\n');
            }
  • The variable x must be an iterable value.

WHILE

Loops over a block of code as long as a specific condition is true.

Example and Uses

          
            let i, text;
            while ( i < 10) {
              text += "The number is " + i;
              i++;
            }
  • Don't forget to increase the variable in the condition or there will be an infinite loop and will crash the browser.

DO WHILE

This loop will execute at least once even if the condition is false. If true, then it will execute as long as the condition is true.

Example and Uses

          
            let i, text;
            do {
              text =+ "The number is " + i;
              i++;
            }
            while ( i < 10);
  • Don't forget to increase the variable in the condition or there will be an infinite loop and will crash the browser.
